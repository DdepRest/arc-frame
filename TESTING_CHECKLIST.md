# ✅ Чек-лист ручного тестирования A.R.C. Frame

> **Версия:** 3.24.0 | **Дата:** 17.06.2026
> **Сборка:** `publish\MosquitoNetCalculator.exe`
> Запустите приложение и пройдите каждый пункт. Ставьте `[x]` напротив пройденных.

---

## 1. 🎨 Переключение темы (светлая ↔ тёмная)

- [ ] Найдите кнопку переключения темы → нажмите
- [ ] Все цвета плавно переходят (анимация ~280мс)
- [ ] Строки таблицы меняют фон при наведении корректно в обеих темах
- [ ] Заголовок окна Windows меняет цвет (тёмная тема → тёмный заголовок)
- [ ] Закройте приложение → откройте заново → тема **сохранилась**

## 2. 📐 Расчёт позиций

- [ ] Добавьте «Anwis» (Белый, 1000×1000, 1 шт) → цена 1800,00, площадь 1,000 м², сумма 1800,00
- [ ] Добавьте «ПСУЛ» (1000×2000, 1 шт) → 6,000 м.п. × 100 = 600,00
- [ ] Добавьте «Уплотнение» (Серый, 1000×2000, 1 шт) → 6,000 м.п. × 250 = 1500,00
- [ ] Добавьте «Работа» (цена 5000) → 1 шт. × 5000 = 5000,00
- [ ] Итоговая сумма внизу = сумма всех активных строк

## 3. 🔧 Переключатель монтажа

> **⚠️ Исправлено в v3.21.0:** Mode 2 «В конструкцию» теперь вычитает 500₽ (было: прибавлял 0₽). Заголовок меню исправлен с «В В конструкцию» на «В конструкцию».

- [ ] Для «Anwis»: нажмите на переключатель монтажа 3 раза подряд
  - Зелёный `—` → монтаж включён, сумма без изменений
  - Красный `✕` → **без монтажа**, сумма −500
  - Синий `В` → **в конструкцию**, сумма −500
- [ ] В контекстном меню заголовок показывает **«В конструкцию»** (без дубля «В В»)
- [ ] Для «На навесах»: аналогично, 3 режима работают
- [ ] Для «Отлив» или «ПСУЛ»: переключатель **неактивен** (серый «—»)
- [ ] Введите пользовательскую сумму в поле «Сумма:» → значение применяется к вычету/надбавке

## 4. ↩️ Отмена/Повтор (Ctrl+Z / Ctrl+Y)

- [ ] Добавьте 3 позиции по очереди
- [ ] Нажмите **Ctrl+Z** → последняя позиция исчезает
- [ ] Нажмите **Ctrl+Z** ещё раз → предпоследняя исчезает
- [ ] Нажмите **Ctrl+Y** → отменённая позиция возвращается
- [ ] Индикатор «● Есть изменения» появляется при редактировании и исчезает после сохранения

---

## 5. 💾 Сохранение и загрузка заказов

- [ ] Заполните данные: имя заказчика, телефон, адрес, позиции
- [ ] Нажмите **Ctrl+S** (или кнопку «Сохранить») → тост «Заказ сохранён»
- [ ] Перейдите на вкладку «Заказы» → заказ появился в списке
- [ ] **Закройте** приложение полностью → **откройте** заново
- [ ] На вкладке «Заказы» → дважды кликните по заказу → все данные загрузились
- [ ] Удалите заказ через контекстное меню → он исчезает из списка

- [ ] Снимите галочку → блок скрывается

- [ ] Добавьте ещё одно Доп. КП → суммы суммируются

## 12. 🏭 «Отправить на завод» (Новинка v3.24.0)

> **Назначение:** Быстрое формирование текста с размерами и составом заказа для отправки на производство.

### 12.1 Открытие окна

- [ ] На вкладке «Расчёт» добавьте несколько позиций (Anwis, Отлив, Работа, ПСУЛ)
- [ ] Нажмите кнопку **«На завод»** в верхней панели действий
- [ ] Окно открывается с плавной fade-in анимацией (~250 мс)
- [ ] Заголовок окна: иконка ✉️ + «Отправить на завод» с синей акцентной полосой
- [ ] Окно перетаскивается за заголовок
- [ ] Окно ресайзится (тянуть за край/угол)

### 12.2 Умный выбор по умолчанию

- [ ] **Производственные товары (Anwis, На навесах, Отлив, Козырёк)** — галочки **включены**; **Короб, Уплотнение, Откос** — галочки **выключены**
- [ ] **Непроизводственные позиции (Работа, ПСУЛ, Доставка, Брус, Пояс)** — галочки **сняты**
- [ ] **Доп. КП** — галочки **включены**
- [ ] Счётчик в чипе показывает: `Выбрано: X из Y` (где Y = общее число позиций)

### 12.3 Переключение чекбоксов

- [ ] Кликните на чекбокс **один раз** — галочка переключается мгновенно (без двойного клика)
- [ ] При включении/выключении:
  - Счётчик `Выбрано: X из Y` **обновляется**
  - Текст в **предпросмотре** обновляется
  - Счётчик строк и символов обновляются

### 12.4 Кнопки «Выбрать все» / «Снять все»

- [ ] Нажмите **«Выбрать все»** → все галочки включены, счётчик = `Выбрано: Y из Y`
- [ ] Нажмите **«Снять все»** → все галочки сняты, счётчик = `Выбрано: 0 из Y`
- [ ] Предпросмотр при 0 выбранных — пустой (счётчик строк = 0, символов = 0)

### 12.5 Формат текста в предпросмотре

- [ ] Заполните поле **«Адрес»** в боковой панели заказчика (например, «ул. Ленина, 15»)
- [ ] В предпросмотре первая строка: `Адрес: ул. Ленина, 15`
- [ ] Если адрес не заполнен — строка «Адрес:» **отсутствует**
- [ ] Если есть выбранные Доп. КП с номерами — строки вида `К КП № 2-1`
- [ ] Товары **сгруппированы по названию** с заголовком-именем и двоеточием:
  ```
  Anwis:
  1000×1000 — 1 шт.
  1200×1400 — 2 шт.

  Отлив:
  200×1000 — 1 шт.
  ```
- [ ] Для штучных товаров (Работа, Доставка) формат: `1 шт.` (без размеров)
- [ ] Шрифт в предпросмотре — **моноширинный** (Consolas)

### 12.6 Счётчики предпросмотра

- [ ] В заголовке предпросмотра показано количество строк: `(N стр.)`
- [ ] В футере предпросмотра показано количество символов: `Символов: N`
- [ ] При пустом предпросмотре: `Символов: 0`, счётчик строк скрыт

### 12.7 Копирование в буфер обмена

- [ ] Нажмите кнопку **«Скопировать»** (или иконку 📋)
- [ ] Тост: «Текст скопирован в буфер обмена!» (зелёный)
- [ ] Вставьте (Ctrl+V) в Блокнот → текст совпадает с предпросмотром
- [ ] **Без выбранных позиций:** нажмите «Скопировать» → тост «Нет выбранных позиций.» (жёлтый)

### 12.8 Закрытие окна

- [ ] Нажмите **✕** в заголовке → окно закрывается
- [ ] Нажмите кнопку **«Закрыть»** в футере → окно закрывается
- [ ] Нажмите клавишу **Escape** → окно закрывается (кнопка «Закрыть» = IsCancel)
- [ ] При наведении на ✕ — плавный красный оверлей (150 мс CubicEase), значок становится белым

### 12.9 Интеграция с Доп. КП

- [ ] Добавьте 2 Доп. КП с номерами (например, «2-1» и «2-2»)
- [ ] Откройте «На завод» → Доп. КП в секции **«ДОПОЛНИТЕЛЬНЫЕ КП»**
- [ ] В предпросмотре строки: `К КП № 2-1`, `К КП № 2-2`
- [ ] Отключите одно Доп. КП → строка исчезает из предпросмотра

## 11. 🖥️ Максимизация окна (ИСПРАВЛЕНО в v3.21.0)

> **⚠️ Исправлено в v3.21.0:** Окно больше не перекрывает панель задач Windows при развёртывании.

- [ ] Нажмите кнопку развёртывания (□) в заголовке окна
- [ ] Окно занимает весь экран, **но панель задач ВИДНА** внизу
- [ ] Нажмите кнопку снова → окно возвращается к нормальному размеру
- [ ] Дважды кликните по заголовку → то же самое поведение
- [ ] Перетащите заголовок развёрнутого окна → окно сворачивается и следует за мышью
- [ ] На **двойном мониторе**: разверните окно на каждом мониторе → панель задач не перекрывается
