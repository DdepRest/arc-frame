## 5. 🔧 Переключатель монтажа

> **⚠️ Исправлено в v3.21.0:** Mode 2 «В конструкцию» теперь вычитает 500₽.  
> **⚠️ Исправлено в v3.37.1:** Вычет/надбавка монтажа теперь умножается на `Quantity` (per piece).

### 5.1 Базовые режимы

- [ ] Для «Anwis»: нажмите на переключатель монтажа 3 раза подряд
  - Зелёный `—` → монтаж включён, сумма без изменений
  - Красный `✕` → **без монтажа**, сумма −500
  - Синий `В` → **в конструкцию**, сумма −500
- [ ] В контекстном меню заголовок показывает **«В конструкцию»** (без дубля «В В»)
- [ ] Для «На навесах»: аналогично, 3 режима работают
- [ ] Для «Отлив» или «ПСУЛ»: переключатель **неактивен** (серый «—»)
- [ ] Введите пользовательскую сумму в поле «Сумма:» → значение применяется к вычету/надбавке

### 5.2 Монтаж с Quantity > 1 (v3.37.1)

- [ ] Добавьте Anwis 1000×1000 × **3 шт.** × цена 1800 → итоговая сумма без монтажа = **5248,80 ₽**
- [ ] Переключите в режим **«Без монтажа»** (красный ✕) → вычет = **1500 ₽** (500 × 3), итог = **3748,80 ₽**
- [ ] Переключите в режим **«В конструкцию»** (синий В) → вычет = **1500 ₽**, итог = **3748,80 ₽**
- [ ] Tooltip при наведении на переключатель показывает `руб./шт. × Кол-во`
- [ ] Для Quantity = 1 поведение не изменилось (вычет 500 ₽, итог 1249,60 ₽)

## 14. 🔄 Автообновление (Новинка v3.37.0)

> **Назначение:** Автоматическая проверка и установка обновлений через GitHub Releases.

### 14.1 Диалог при запуске

- [ ] При запуске: если доступно обновление → показывается **диалог** (не toast)
- [ ] Диалог содержит список изменений (changelog) с момента установленной версии
- [ ] В changelog видны бейджи типа (Новинка / Улучшение / Исправление)
- [ ] Кнопки: **«Отмена»** и **«Скачать и установить»**

### 14.2 TitleBar-полоска прогресса

- [ ] Нажмите «Скачать и установить» → в TitleBar появляется **тонкая полоска** (3px) с цветом Accent (fade-in ~200 мс)
- [ ] Полоска плавно заполняется слева направо (% скачивания)
- [ ] После завершения скачивания полоска исчезает (fade-out ~250 мс) и приложение **перезапускается**
- [ ] Если закрыть диалог во время скачивания → полоска остаётся видна, загрузка продолжается

### 14.3 Ручная проверка

- [ ] Меню **Настройки** → «Проверить обновления»
- [ ] Toast: «Проверка наличия обновлений...»
- [ ] Если обновлений нет → toast **«Обновлений нет ✓»** (зелёный)
- [ ] Если есть → тот же диалог с changelog

### 14.4 Toast-фильтрация

- [ ] При **автоматической** проверке на старте: нет toast'а «Обновлений нет»
- [ ] При **ручной** проверке: toast «Обновлений нет ✓» показывается
- [ ] Ошибки сети при авто-проверке → молчаливый выход (не блокирует UI)

## 15. 🏭 «Отправить на завод»

> **Назначение:** Быстрое формирование текста с размерами и составом заказа для отправки на производство.

### 15.1 Открытие окна

- [ ] На вкладке «Расчёт» добавьте несколько позиций (Anwis, Отлив, Работа, ПСУЛ, Брус)
- [ ] Нажмите кнопку **«На завод»** в верхней панели действий
- [ ] Окно открывается с плавной fade-in анимацией (~250 мс)
- [ ] Заголовок окна: иконка ✉️ + «Отправить на завод» с синей акцентной полосой
- [ ] Окно перетаскивается за заголовок
- [ ] Окно ресайзится (тянуть за край/угол)

### 15.2 Умный выбор по умолчанию

- [ ] **Производственные товары (Anwis, На навесах, Козырёк)** — галочки **включены**; **Отлив, Короб, Уплотнение, Откос** — галочки **выключены**
- [ ] **Непроизводственные позиции (Работа, ПСУЛ, Доставка, Брус, Пояс)** — галочки **сняты**
- [ ] **Доп. КП** — галочки **включены**
- [ ] Счётчик в чипе показывает: `Выбрано: X из Y` (где Y = общее число позиций)

### 15.3 Переключение чекбоксов

- [ ] Кликните на чекбокс **один раз** — галочка переключается мгновенно (без двойного клика)
- [ ] При включении/выключении:
  - Счётчик `Выбрано: X из Y` **обновляется**
  - Текст в **предпросмотре** обновляется
  - Счётчик строк и символов обновляются

### 15.4 Кнопки «Выбрать все» / «Снять все»

- [ ] Нажмите **«Выбрать все»** → все галочки включены, счётчик = `Выбрано: Y из Y`
- [ ] Нажмите **«Снять все»** → все галочки сняты, счётчик = `Выбрано: 0 из Y`
- [ ] Предпросмотр при 0 выбранных — пустой (счётчик строк = 0, символов = 0)

### 15.5 Формат текста в предпросмотре

- [ ] Заполните поле **«Адрес»** в боковой панели заказчика (например, «ул. Ленина, 15»)
- [ ] В предпросмотре первая строка: `Адрес: ул. Ленина, 15`
- [ ] Если адрес не заполнен — строка «Адрес:» **отсутствует**
- [ ] Если есть выбранные Доп. КП с номерами — строки вида `К КП № 2-1`
- [ ] Товары **сгруппированы по названию** с заголовком-именем и двоеточием:
  ```
  Anwis:
  1000×1000 — 1 шт.
  1200×1400 — 2 шт.

  Отлив:
  200×1000 — 1 шт.
  ```
- [ ] Для штучных товаров (Работа, Доставка) формат: `1 шт.` (без размеров)
- [ ] Для AmountOnlyProducts (Брус, Пояс) формат: только количество и сумма (без габаритных размеров)
- [ ] Шрифт в предпросмотре — **моноширинный** (Consolas)

### 15.6 Счётчики предпросмотра

- [ ] В заголовке предпросмотра показано количество строк: `(N стр.)`
- [ ] В футере предпросмотра показано количество символов: `Символов: N`
- [ ] При пустом предпросмотре: `Символов: 0`, счётчик строк скрыт

### 15.7 Копирование в буфер обмена

- [ ] Нажмите кнопку **«Скопировать»** (или иконку 📋)
- [ ] Тост: «Текст скопирован в буфер обмена!» (зелёный)
- [ ] Вставьте (Ctrl+V) в Блокнот → текст совпадает с предпросмотром
- [ ] **Без выбранных позиций:** нажмите «Скопировать» → тост «Нет выбранных позиций.» (жёлтый)

### 15.8 Закрытие окна

- [ ] Нажмите **✕** в заголовке → окно закрывается
- [ ] Нажмите кнопку **«Закрыть»** в футере → окно закрывается
- [ ] Нажмите клавишу **Escape** → окно закрывается (кнопка «Закрыть» = IsCancel)
- [ ] При наведении на ✕ — плавный красный оверлей (150 мс CubicEase), значок становится белым

### 15.9 Интеграция с Доп. КП

- [ ] Добавьте 2 Доп. КП с номерами (например, «2-1» и «2-2»)
- [ ] Откройте «На завод» → Доп. КП в секции **«ДОПОЛНИТЕЛЬНЫЕ КП»**
- [ ] В предпросмотре строки: `К КП № 2-1`, `К КП № 2-2`
- [ ] Отключите одно Доп. КП → строка исчезает из предпросмотра
