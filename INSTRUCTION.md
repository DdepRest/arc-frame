# A.R.C. Frame — MosquitoNetCalculator v3.24.0

Программа для быстрого расчёта москитных сеток, формирования КП и ведения заказов.

---

## 🔹 Быстрый старт

### 1. Создать заказ
На вкладке **Расчёт** заполните поля:

| Поле | Что вводить |
|------|------------|
| **Тип** | Anwis, На навесах, Отлив, ПСУЛ, Уплотнение, Работа и др. |
| **Цвет** | Доступные цвета для выбранного типа |
| **Ширина, мм** | Размер изделия |
| **Высота, мм** | Размер изделия |
| **Кол-во** | Обычно 1 |
| **Цена** | Подставляется из прайса, можно изменить вручную |

→ Нажмите **Enter** или **«Добавить»**

### 2. Сохранить
**Ctrl+S** или кнопка **«💾 Сохранить»**

### 3. Распечатать
Кнопка **«🖨 Печать КП»** → предпросмотр → системный диалог печати / PDF

---

## 🔹 Вкладки

| Вкладка | Назначение |
|---------|-----------|
| **Расчёт** | Основная. Быстрый ввод, таблица позиций, итоги |
| **Цены** | Редактирование прайс-листа (клик по ячейке → сохранить) |
| **Заказы** | Журнал сохранённых заказов, импорт/экспорт |
| **Обновления** | История версий программы |

---

## 🔹 Таблица позиций

- **Галочка ☑** — включить/исключить позицию из итога (не удаляя)
- **Ширина, Высота, Кол-во, Цена** — редактируются двойным кликом
- **✕** — удалить строку
- **Монтаж** — переключатель для Anwis и На навесах:
  - `─` (зелёный) — монтаж включён
  - `✕` (красный) — без монтажа (−500 ₽)
  - `В` (синий) — в конструкцию (−500 ₽)
  - Правый клик по кнопке — изменить сумму вычета

---

## 🔹 Боковая панель (слева)

- **Заказчик** — имя, телефон, адрес
- **Договор** — номер (префикс−число), дата, статус
- **Примечания** — свободный текст
- **Доп. КП** — дополнительные КП (галочка в заголовке скрывает блок)

---

## 🔹 Горячие клавиши

| Клавиша | Действие |
|---------|---------|
| **Ctrl+S** | Сохранить заказ |
| **Ctrl+Z** | Отменить |
| **Ctrl+Y** | Повторить |
| **Enter** | Добавить позицию |

---

## 🔹 Советы

- **Поиск** — начните печатать название товара, подсказки появятся автоматически
- **Индикатор «● Есть изменения»** — оранжевый чип вверху, пока не сохранено
- **Подтверждение закрытия** — если есть несохранённые изменения, программа спросит
- **Префикс договора** — сохраняется между запусками, можно изменить в боковой панели
- **Темы** — кнопка **☀/🌙** в правом верхнем углу (сохраняется между запусками)
