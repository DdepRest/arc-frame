================================================
 MosquitoNetCalculator — Сборка установщика
================================================

Для создания установщика (.exe) требуется:
  - Inno Setup 6.0 или новее (скачать: https://jrsoftware.org/isdl.php)
    ВНИМАНИЕ: используются функции RegGetValueNames и TStringList.Delimiter,
    доступные только в Inno Setup 6.0+ (проверить версию: Help → About).
  - Собранный проект в папке publish\

Порядок сборки:

  1. Запустите build.bat — он создаст папку publish\ со всеми файлами
     (самодостаточный .exe с встроенным .NET 8 — .NET устанавливать отдельно НЕ нужно)

  2. Откройте installer.iss в Inno Setup

  3. Нажмите Build → Compile (или Ctrl+F9)

  4. Готовый установщик появится в папке Output\:
       Output\SetupMosquitoNetCalculator-3.24.0.exe

Что делает установщик:
  - Позволяет выбрать папку установки
  - Создаёт ярлык в меню Пуск
  - Опционально — ярлык на рабочем столе
  - Проверяет версию Windows (требуется Win10 1809 / Win11)
  - Проверяет наличие WebView2 Runtime (>= 100.0.0.0) — нужен для предпросмотра КП
  - Проверяет наличие Visual C++ Redistributable 2015-2022 (>= 14.30)
  - Если какой-то компонент отсутствует или устарел, установщик
    АВТОМАТИЧЕСКИ скачает и установит ТОЛЬКО недостающие компоненты.
    Если в системе уже есть всё нужное — НИЧЕГО не скачивается и не устанавливается.
  - .NET 8 Desktop Runtime уже встроен в программу (self-contained), его устанавливать не нужно.
    Состояние .NET Desktop Runtime отображается только для диагностики.

================================================
 Предварительная проверка зависимостей
================================================

Прежде чем запускать установщик, можно проверить состояние системы
и при необходимости доустановить недостающие компоненты.

Файлы:
  - check-deps.bat  — самодостаточный скрипт проверки (встроенный PowerShell, не требует внешних файлов)

Использование:
  check-deps.bat               — проверить и показать отчёт
  check-deps.bat -Install      — проверить и автоматически доустановить недостающее
                                  (через winget, либо прямую загрузку)
  check-deps.bat -Quiet         — без информационных сообщений (только ошибки)
  check-deps.bat -Json          — вывод в формате JSON (для CI/CD)

Скрипт проверяет:
  - Версию Windows (Build >= 17763 = Windows 10 1809 / 11)
  - .NET 8 Desktop Runtime (только для диагностики — программа самодостаточная)
  - WebView2 Runtime (>= 100.0.0.0)
  - VC++ Redistributable 2015-2022 (x64, >= 14.30)

Скрипт лежит в корне проекта (рядом с publish/) и автоматически
копируется в publish/ скриптом build.bat — после сборки он попадает
в инсталлятор. После установки программы он окажется в её каталоге ({app}),
и пользователь может запускать его там для диагностики / автоустановки зависимостей.
